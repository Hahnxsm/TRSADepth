# TRSADepth: Self-Supervised Monocular Depth Estimation with Texture-Robust Sparse Attention in Natural Environments

## Train
```shell
python train.py
--model_name [model_name] # 
--split [eigen_zhou]
--dataset [kitti]
--pose_model_type [resnet]
--data_path [data_path]
--frame_ids 0 -1 1
--batch_size [8/16]
--png
--height [384]
--width [640]
--num_epochs [15/20/...]
--log_dir [./runs/log_dir]
```
## Evaluation
```shell
python [eval.py]
--eval_split [eigen] 
--dataset [kitti]
--data_path [data_path]
--load_weights_folder [...]
--eval_mono
```
